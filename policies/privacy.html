Cesium Privacy Policy

Cesium holds privacy in the highest respect. I value the privacy of all users.

This website, Cesium, has no tracking mechanisms whatsoever. There are some cookies in use that only allow the end user to experience better browsing performance. These cookies are issued by CloudFlare, a service I use to keep my website stable and secure. For more information on their cookies, please visit their website.

Cesium Cloud Privacy Policy

The Cesium Cloud is run with the same privacy-respecting ideology. While I do have a way to access the information you upload to the Cesium Cloud, I refrain from looking at/for this information except where outlined in the Cesium Cloud Terms of Use.


Source code for this website as well as the Cesium Cloud are avaliable on GitHub.
