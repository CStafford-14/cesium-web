The Cesium Cloud

Private. Secure. Trustworthy.
Perfect.

The Cesium Cloud is a Nextcloud instance. Nextcloud is a free-as-in-freedom and open source server software capable of many things. On the Cesium Cloud, users have access to 10 GB of free cloud storage, Word, Spreadsheet, and Presentation editors, voice/video/screen calling, text chat, and much more!

It's Nextcloud!

Nextcloud is powerful, and with its modern UI called the "Nextcloud Hub," users are surely able to work seamlessly. Learn more about Nextcloud Hub.

Private. Yet Functional.

The Cesium Cloud was created to get away from prying eyes. Data stored on the cloud is locked behind the user's account and is only accessible to their account. That doesn't mean the cloud is skimping on any services...
The cloud is capable of running a full-blown file storage and sharing service as well as providing a convenient and intuitive UI for doing so.

The cloud is secure by design, so you won't have to worry about attackers. Accounts can be created by oAuth or Email/Password and secured by a TOTP authenticator app or a hardware security key.
